# Objective

Implement the foundation layer in `cli_replay/session.py`: data types (TypedDicts), validation functions, and write functions for the `.clirec` JSON Lines format. Also create `tests/conftest.py` with shared test fixtures.

# Implementation Plan

1. Define `SessionHeader` and `SessionEvent` TypedDicts
2. Implement `validate_header(data)` — raises `ValueError` on missing fields, wrong types, unknown version
3. Implement `validate_event(data)` — raises `ValueError` on missing fields, invalid type value
4. Implement `write_header(f, header)` — serialize as JSON line + newline + flush
5. Implement `write_event(f, event)` — serialize as JSON line + newline + flush
6. Create `tests/conftest.py` with `sample_header`, `sample_events`, `tmp_clirec` fixtures

# Acceptance Criteria

- [x] `SessionHeader` TypedDict with version, timestamp, width, height fields
- [x] `SessionEvent` TypedDict with t, type, data fields
- [x] `validate_header` rejects missing fields, wrong types, version != 1
- [x] `validate_event` rejects missing fields, type not in ("i", "o")
- [x] `write_header` writes valid JSON followed by newline, flushes
- [x] `write_event` writes valid JSON followed by newline, flushes
- [x] `tests/conftest.py` provides reusable fixtures
- [x] All tests pass with `pytest tests/test_session.py`
- [x] `bpsai-pair arch check cli_replay/session.py` passes
